路由說明

路由引擎依 優先序（數字小=優先）逐條比對告警資料。
匹配欄位支援：resourceName、alertName、criticality。
匹配模式使用 glob 語法：custA-*（前綴）、*CPU*（包含）、CRITICAL（完全比對）。
第一條匹配的規則生效，未匹配則通知「預設群組」。
